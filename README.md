# 🚀 Comprehensive Business Website (React + Modern Frontend)

## 📌 Project Overview

This project is a modern multi-page business website built using React, advanced CSS (Flexbox & Grid), and modern JavaScript (ES6+). The website demonstrates responsive design, reusable components, accessibility best practices, and performance optimization techniques.

---

## 🎯 Objectives

- Build 5+ responsive pages
- Implement modern CSS Grid & Flexbox layouts
- Use React functional components with hooks
- Implement dark/light theme toggle
- Add form validation
- Ensure accessibility compliance
- Optimize performance

---

## 🛠️ Technologies Used

- React (Vite)
- React Router DOM
- JavaScript ES6+
- CSS3 (Grid & Flexbox)
- Responsive Design
- Contextual Theme State
- Accessibility (ARIA basics)
